# StudyMate LLM Capstone

This folder contains all capstone material for the final project.

## Start Here

Students should start in `studymate_llm/`.

- Read `studymate_llm/README.md` for the overview.
- Read `studymate_llm/EXERCISE_GUIDE.md` for the required tasks.
- Run the student project from `studymate_llm/`.

## Folder Guide

- `studymate_llm/`: the student-facing capstone folder
- `studymate_llm/reference_solution/`: the recommended reference solution kept inside the same project folder

## Recommended Reading Order

1. `studymate_llm/README.md`
2. `studymate_llm/EXERCISE_GUIDE.md`
3. `studymate_llm/app.py`
4. `studymate_llm/tests/`
5. `studymate_llm/reference_solution/` only after attempting the exercise

## Recommendation

Use `studymate_llm/reference_solution/` as the main reference solution going forward.
