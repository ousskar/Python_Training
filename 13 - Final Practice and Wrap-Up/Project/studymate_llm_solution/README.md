# StudyMate LLM Solution Project

This folder contains a completed reference implementation for the StudyMate LLM capstone exercise.

## Status

This is a legacy duplicate solution folder.
The preferred and easier-to-follow solution now lives in `Project/studymate_llm/reference_solution/`.

## What Is Solved Here

- summary generation
- flashcard generation
- quiz generation
- environment-based LLM stub usage
- session saving
- example tests for the completed behavior

## How To Run

1. Open a terminal in `13 - Final Practice and Wrap-Up`.
2. Run `python -m Solutions.studymate_llm_solution.app`.
3. Run `pytest Solutions/studymate_llm_solution/tests`.

## Notes For Instructors

- The student project remains in `studymate_llm`.
- This folder is kept for compatibility, not as the primary reference path.
