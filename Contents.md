# Python Class Plan

- **Duration:** 3 days
- **Topic:** Python

## Day 1 - Python Foundations

**Theme:** Getting comfortable with the language basics

### Introduction to Python

- Overview of Python
- Setting up the Python environment
- Writing and executing Python scripts

### Data Types and Variables

- Data types: Numbers, Strings, Lists, Tuples, Booleans, Dictionaries
- Variables: naming conventions and assigning values

### Iterables, Indexing, and Slicing

- Understanding iterables
- Using iterators
- Accessing elements in sequences
- Slicing lists and strings

### Flow Control

- If-Else statements
- For loops
- Enumerate and zip
- Continue
- Initializing variables
- Single-line loops (list comprehension)
- While loops

### Practice Session

- Function error checking and handling
- Exercises on variables, collections, loops, and slicing
- Mini project: build a small command-line program that filters and formats a list of names or numbers

## Day 2 - Functions and Code Organization

**Theme:** Writing reusable and structured code

### Functions

- Inputs and outputs
- Getting help on functions
- Creating functions: defining and calling
- Global and local variable scope
- Copies and references of variables

### Working with Files

- Opening and closing files
- Reading from text files
- Writing and appending to files
- Using `with` for safe file handling
- Working with file paths

### Python Libraries, Modules, and Packages

- Importing and using Python libraries
- Creating and using modules
- Understanding packages
- Managing namespaces

### Object-Oriented Programming

- Defining classes and objects
- Inheritance and polymorphism

### Higher-Order Functions and Decorators

- Higher-order functions
- Introduction
- Map
- Closures
- Introduction to decorators
- Functions as first-class objects
- Writing basic decorators
- Using the `@decorator` syntax
- Nesting and chaining decorators
- Using `functools.wraps`

## Day 3 - Testing, Debugging, and Reliability

**Theme:** Writing code you can trust

### Unit Testing

- Writing test cases
- Using the `unittest` framework
- Mocking dependencies in tests
- Writing and running tests with `pytest`
- Guided practice writing, fixing, and improving tests

### Debugging

- Debugging fundamentals and common Python error types
- Handling errors with `try`, `except`, `else`, and `finally`
- Using `print()` and built-in helpers such as `type()`, `id()`, `dir()`, `vars()`, and `help()`
- Python debugging tools: `pdb`, `breakpoint()`, IDE debuggers, logging, and assertions
- Guided debugging practice and troubleshooting exercises

### Final Practice and Wrap-Up

- 100 cumulative exercises across foundations, functions, files, OOP, testing, and debugging
- Integrated exercises combining functions, classes, decorators, testing, and debugging
- Final review and Q&A
