# StudyMate LLM Reference Solution

This folder contains the completed reference solution for the StudyMate LLM capstone.

This is the preferred solution location because it stays next to the student starter, tests, and exercise guide.

## Folder Layout

- `..`: student starter project, exercise guide, tests, and data
- `reference_solution/`: completed reference implementation

## How To Run

1. Open a terminal in `13 - Final Practice and Wrap-Up`.
2. Run `python -m studymate_llm.reference_solution.app`.
3. Run `pytest studymate_llm/reference_solution/tests`.

## Purpose

Students should work in the main `studymate_llm` folder.
Instructors can use this subfolder as the completed reference implementation.
